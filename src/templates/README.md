# Templates - Ingelas Design

**VARNING: LÅS PÅ ALLA FILER HÄR!**

Denna katalog innehåller mallar skapade av vår Art Director Ingela via Figma.

## Regler

1. **Ingen får ändra mallarna** - Filerna här är skrivskyddade för alla utom Ingela
2. **Endast läsrättighet** - Du får läsa mallarna för att förstå vilka placeholders som finns
3. **Användning** - Använd mallarna som de är i dina sidor/komponenter
4. **Ändringar** - Om något behöver ändras i en mall, kontakta Ingela

## Tillgängliga Mallar

### `Desktop_16x9_header.tsx`
- **Källa**: Figma node `2002:2`
- **Skapad**: 2026-02-04
- **Placeholders**: 
  - `image` - Hero bakgrundsbild
  - `title` - Huvudrubrik (H1)
  - `content` - Brödtext under titel
  - `cta_text` - CTA-knapptext
  - `cta_url` - CTA-knapplänk
  - `subtitle` - Underrubrik (rows)
  - `heading` - Sekundär rubrik (rows)
  - `body` - Brödtext (rows)
  - `row2_image` - Bild höger sida
  - `row3_image` - Bild vänster sida
  - `cards_title` - Kort-sektion rubrik
  - `card1`, `card2`, `card3` - Kort-innehåll
